<img src="Image/1.png"><img src="Image/2.png"><img src="Image/3.png">
## Requirements

- Node.js
- npm

## Install

```bash
npm install
```

## Run

```bash
npm start
```

The server starts on:

```text
http://localhost:3000
```

To use a different port:

```bash
PORT=8080 npm start
```
